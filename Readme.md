# 🤖 AI HR Screening Agent (n8n)

An automated AI-powered HR assistant built using n8n.

## 🚀 Features
- Resume parsing
- AI-generated screening questions
- ATS scoring
- Automated Gmail feedback
- Fully automated workflow

## 🛠 Tech Stack
- n8n
- LLM (Gemini/OpenAI)
- Gmail API
- JavaScript

## 📥 How to Use

1. Install n8n
2. Import `ai-hr-screening-workflow.json`
3. Configure credentials
4. Activate workflow

See `.env.example`
